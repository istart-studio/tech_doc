----
主机标识符：
主机标识符全为0 本机
主机标识符全为1 广播
----

----
无等级域间路由 CIDR:
更好的划分网络
例如：10.100.122.2/24，24位进行前后划分，前面24位是网络号，后面8位是主机号

广播地址：10.100.122.255
子网掩码：255.255.255.0

网络号：将IP与子网掩码 AND 计算，就是网络号。
----

举例：16.158.165.91/22
16.158.<101001><01>.91

网络号 16.158.<101001>

主机号 <01>.91

第一个地址是 16.158.<101001><00>.1 16.158.164.1

子网掩码 255.255.<111111><00>.0

广播地址 16.158.<101001><00>.0
